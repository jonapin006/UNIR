Laboratorio de sistemas operativos

Este es el documento anexo del script desarrollado para la materia de sistemas operativos, el cual fue creado utilizando el comando vim de la siguiente manera:

  1. Se debe añadir "#!/bin/bash", el cual le indica al sistema operativo que invoque su shell específico para ejecutar los comandos que vienen a continuación del script
  2. Luego de esto, añadí el comando "echo" el cual imprime un mensaje en pantalla: "Este script se desarrolló para instalar y configurar un servidor web Apache, debes digitar la clave de sudo para el correcto funcionamiento del script"
  3. Luego, se actualizan los repositorios de linux
  4. Seguido de esto, se instala Apache como servidor web
  5. Se hace al descarga del archivo html, el cual es el que se está visualizando, con el paso a paso de la creación del script
  6. Como Apache tiene un archivo index.html por defecto, lo renombro y ubico el archivo descargado en la ruta pública de Apache "/var/www/html/"
  7. Abre el navegador para que este archivo sea visualizado
